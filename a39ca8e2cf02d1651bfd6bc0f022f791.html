《我和我的十七歲》片頭曲柯有倫量身製作甜蜜K歌

由種子音樂量身打造音樂製作、東森自製的最新偶像劇《我和我的十七歲》將於臺視4月9日晚上10點首播。本劇自開拍後就造成不小的轟動，在開播前臉書粉絲頁就吸引了近十萬人關註。不止有堅強的卡司：李國毅、謝欣穎、王家梁、周曉涵、鄭茵聲領銜主演，東森電視與種子音樂聯合統籌量身訂做，由品冠、柯有倫、林明禎、林芯儀、井柏然等K歌藝人齊聚包辦了全劇的原聲帶。

《我和我的十七歲》在播出前就已曝光不少宣傳短片，其中搭配的音樂更是引起不少網友熱烈詢問。片頭曲特地邀請實力派時尚型男柯有倫Alan，及大馬國民女神林明禎，獻唱〈My Only Love〉。柯有倫自2014年的專輯《解放浪漫》後一年內已四度為戲劇及電影獻聲。柯有倫主演的電視劇《鋼鐵之心》精神指定曲〈兄弟還記得嗎〉，與邱凱偉Darren兄弟合唱，成為新一代兄弟、友誼的代表作。電影《菜鳥》與韓國戲劇女神崔允素跨國合唱的〈為愛〉。以及香港4K網絡劇《天才在左 瘋子在右》柯有倫再度為戲創作擅長及喜愛的搖滾曲風〈A Life〉。音樂創作類型相當的廣，首首膾炙人口，可說是影劇界的影劇熱門寵兒。

這次再次受到《我和我的十七歲》戲劇的青睞，為其戲劇量身製作的〈My Only Love〉是首流行搖滾的輕快曲風。劇里因描述十七歲時的青春愛情回憶，那種愛又不敢說出口的酸甜，越是喜歡對方就越是愛捉弄對方。為此Alan再配唱時想把一種青春無憂無慮的感覺唱進歌里。Alan回想自己在十七歲時候做什麽事情常常沒有考慮太多，無畏放膽地說做就做，不管後果一群人總是笑笑就過了。身為〈My Only Love〉的製作人，Alan為求完美前後進錄音室耗時兩個禮拜才滿意成果，就為了打造一首專屬戲劇的歌曲。Alan笑稱這比錄製自己的專輯還要辛苦。

近日成為戲劇新星的林明禎才剛拍完《救殭清道夫》就馬上投入錄音表示這是她人生第一次的合唱曲，第一次就能與師兄柯有倫合作，感到非常開心與期待。她表示雖然自己的My only love還沒出現，但是唱著這首歌心裡就覺得很甜，尤其和師兄Alan你一句我一句對唱時，感覺真的就像跟喜歡的人對話般，心底感覺甜蜜蜜。搭配柯有倫磁性嗓音與明禎可愛的甜蜜歌聲，讓人彷佛回到17歲的青春愛情回憶。

甜密K歌〈My Only Love〉從4月8號起即可在各大數位音樂平臺上下載。
